Only for Windows:

Steps to install and run the vital sign detection app:

1. Connect the Infineon radar to the PC via an USB cable.

2. Double click the run_setup.bat file and wait for the installation to finish, which takes a couple of minutes

3. Double click the run_gui.bat

Note: 

1. Internet access is required for the installation only.

2. The setup steps only need to be run once. To run the app after setup, only step 1 and step 3 are required.

3. The app is only designed and tested on Windows 10 64 bit, but it should work for other Windows 64 bit computers.
